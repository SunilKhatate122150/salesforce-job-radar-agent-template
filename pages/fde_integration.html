How do Platform Events connect to Agentforce architecture?CONCEPT
  • Agent → Platform Event: Action publishes event to external systems. Agent confirms payoff → publishes Payoff_Request__e → LOS picks it up
  • Platform Event → Agent: External system publishes event → triggers Flow → updates CRM → affects agent grounding
  • Data Cloud Streaming: Platform Events ingested as streaming engagement events → enriches unified profile real-time
  • YOUR experience: Your LOS integration via Platform Events is the EXACT pattern for Agentforce – external system communication
What are Data Cloud Triggered Flows? How different from CRM triggers?CONCEPT
  • Fire when Data Cloud data changes — similar to record-triggered flows but for DMO events
  • Triggers: DMO created/updated, Calculated Insight crosses threshold, streaming event received
  • Key difference: Operate on UNIFIED data — see the cross-system picture, not just one CRM object
  • Credit score drops below 620 → create Risk Alert task in CRM
  • Identity Resolution merges new record → trigger welcome journey
Key distinction

"CRM triggers see one system's data. Data Cloud triggers see the unified, cross-system truth."
